Meus Projetos

Cadastro de Funcionário

Este projeto consiste em um sistema de cadastro de funcionários, voltado para empresas que desejam organizar e gerenciar informações sobre seus colaboradores. O formulário permite a coleta de dados como nome completo, cargo, departamento, data de admissão, salário e outras informações relevantes, facilitando a gestão de recursos humanos.

Você pode visualizar o código-fonte desse projeto no meu repositório do GitHub: GitHub - Cadastro de Funcionário

Projeto de Reciclagem Criativa

Este projeto é uma iniciativa para promover a reciclagem criativa de materiais que normalmente seriam descartados. Através de oficinas e tutoriais, ensinamos como reutilizar itens como garrafas plásticas e papelão para criar novos objetos e decoração.

A reciclagem é fundamental para a preservação do meio ambiente. Ao reciclar, reduzimos o desperdício, economizamos recursos naturais e ajudamos a diminuir a poluição.

Você pode acompanhar mais sobre esse projeto em minhas redes sociais ou visualizar o código-fonte no meu repositório do GitHub: GitHub - Reciclagem Criativa
